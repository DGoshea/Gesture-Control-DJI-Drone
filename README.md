# Gesture-Control-DJI-Drone
Enables users to control their DJI Drone with hand gestures. 

# Components 
OpenCV for Computer Vision 
Google's MediaPipe for hand tracking and gesture classification
DJITelloPy for interacting with DJI drone API 

# Features
Takeoff, move up, move down, move left, move right, and land with hadn gestures 

# Installs 
pip install opencv-python mediapipe djitellopy numpy
